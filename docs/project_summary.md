# Ecommerce Fraud Detection Analysis

## Objective
Analyze ecommerce transactions to identify fraud patterns using SQL, Python, and Power BI.

## Tools Used

- Python
- Pandas
- NumPy
- SQL
- Power BI
- Jupyter Notebook

## Dataset

- 70,000+ transactions
- Customer information
- Payment methods
- Product categories
- Fraud labels

## Key Insights

- Fraud rate by payment method
- Fraud by product category
- High-risk transaction amounts
- Customer behavior analysis
- Time-based fraud patterns

## Business Impact

The project demonstrates how data analytics can help organizations detect fraud early and reduce financial losses.
